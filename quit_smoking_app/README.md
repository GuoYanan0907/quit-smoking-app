# 这个APP能让你戒烟

> 21天修仙戒烟计划 - 踏上戒烟修仙之路，历经二十一重境界，涤荡烟毒，重铸仙体。

## 项目简介

这是一个帮助吸烟者戒烟的APP，采用修仙主题设计，将21天戒烟计划比喻为修仙之路。

### 核心理念

- **可视化**：直观展示身体恢复进度
- **科学数据**：基于公开医学资料推算
- **游戏化**：修仙境界、成就系统、心魔挑战
- **实用工具**：烟瘾发作时的排解方法

## 功能特性

### 1. 二十一重境界
- 感气境 → 引气境 → 炼气境 → ... → 大罗金仙
- 每天突破一个境界
- 中途放弃则散尽修为

### 2. 身体恢复可视化
- 烟毒（尼古丁）清除进度
- 浊气（一氧化碳）清除进度
- 灵气（血氧）恢复进度
- 气海（肺功能）恢复进度
- 六识（味觉/嗅觉）恢复进度

### 3. 心魔系统
- 心魔来袭（烟瘾发作）
- 多种抵御方法（吐纳、定心、转念等）
- 心魔等级和强度

### 4. 成就系统
- 境界成就
- 修炼成就
- 特殊成就
- 隐藏成就

### 5. 每日打卡
- 记录当天状态
- 烟瘾程度评分
- 修行日记

## 技术栈

- **框架**：Flutter
- **语言**：Dart
- **状态管理**：Provider / Riverpod
- **本地存储**：SharedPreferences + SQLite
- **动画**：Lottie + Rive
- **图表**：fl_chart

## 数据来源

所有健康数据基于以下公开资料推算：
- 世界卫生组织（WHO）戒烟健康益处报告
- 美国癌症协会（ACS）戒烟后身体恢复时间线
- 美国心脏协会（AHA）心血管健康研究
- 美国肺脏协会（ALA）肺功能恢复数据
- 《新英格兰医学杂志》戒烟相关研究论文
- 《柳叶刀》全球疾病负担研究
